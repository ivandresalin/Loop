# Loop — Gestión Circular de Alimentos

Prototipo interactivo (SPA) de **Loop**, una plataforma B2B SaaS que ayuda a restaurantes a reducir el desperdicio de alimentos mediante IA, gestión de inventario y promociones dinámicas.

Cliente de referencia (datos simulados): **Marea Baja - Marisquería** (Guayaquil, Ecuador).

## Stack
- HTML5, CSS3 (vanilla) y JavaScript (vanilla ES6+), sin frameworks.
- Iconografía: [Lucide Icons](https://lucide.dev/) vía CDN.
- Todo el estado y los datos están simulados (mock data) en `index.html`.

## Módulos
1. Dashboard
2. Inventario Inteligente
3. Predicciones IA
4. Promociones QR
5. Residuos
6. Reportes

## Ver en local
Abre `index.html` directamente en tu navegador, no requiere servidor ni build.

## Publicar en GitHub Pages
1. Ve a **Settings → Pages** en este repositorio.
2. En "Build and deployment", selecciona **Deploy from a branch**.
3. Elige la rama **main** y la carpeta **/ (root)**.
4. Guarda. En unos minutos tu sitio estará en `https://ivandresalin.github.io/Loop/`.
